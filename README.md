# Tenjin News Scraper

A news scraping toolkit for collecting, normalizing, and storing articles from configurable sources.

## Status

Early scaffolding. The project layout, scraper modules, and pipeline are still to be implemented.

## Requirements

- Python 3.11+
- `pip` for dependency management

## Getting Started

```bash
# Clone and enter the repo
git clone https://github.com/aztenjin81/tenjin_news_scraper.git
cd tenjin_news_scraper

# Create a virtual environment
python -m venv .venv
source .venv/bin/activate

# Install dependencies (once requirements.txt exists)
pip install -r requirements.txt
```
